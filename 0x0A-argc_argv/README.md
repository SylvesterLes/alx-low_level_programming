There are at least two arguments to main: argc and argv. The first of these is a count of the arguments supplied to the program and the second is an array of pointers to the strings which are those arguments—itstype is (almost) ‘array of pointer to char’. These arguments are passed to the program by the host system's command line interpreter or job control language. The declaration of the argv argument is often a novice programmer's first encounter with pointers to arrays of pointers and can prove intimidating. However, it is really quite simple to understand.
